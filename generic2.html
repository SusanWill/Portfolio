Website portfolio hosted via GitHub Pages

Building on an Alex the Analyst tutorial, I have taken the Astral HTML website template from HTML5Up and, using coding skills gained through the School of Code, modified the HTML and CSS to create a format for my portfolio.

I have:
• removed the forward arrow from the "Home" page.
• removed the "Twitter" icon and added "LinkedIn" and "Portfolio". The icons are from icons8. When clicked, a new tab opens.
• amended the "Work" title to "Projects" and built separate HTML pages linked to each project. Each page gives an overview of the project with a link to the project held on GitHub. A "Back" link has been added at the bottom of the page which returns to the “Projects” page.
• amended the "Contact" page to an "About" page and removed the form.
